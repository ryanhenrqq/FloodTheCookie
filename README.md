# FloodTheCookie

- Flood the Cookie é um jogo do tipo Click-to-Earn, com um foco grande no script. 

- Em resumo, essa é uma recriação um jogo famoso de clicar em um cookie para ganhar pontos, só que usando linguagem de programação de sites

- Aviso: "Flood the Cookie" é uma inspiração simplificada de Cookie Clicker disponível na Steam, sem intenção nenhuma de lucro
